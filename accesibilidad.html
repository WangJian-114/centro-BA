Experiencia Digital Accesible

Nuestro portal está diseñado bajo los estándares WCAG 2.1 para que la información de transporte llegue a todos, sin importar el dispositivo o la capacidad de navegación.

Navegación por teclado

Toda la interfaz es operable mediante atajos de teclado, con indicadores de foco claros para usuarios con movilidad reducida.

Compatibilidad WCAG

Cumplimos con el nivel AA de las pautas de accesibilidad para el contenido web, garantizando robustez y legibilidad.

Contraste alto

Nuestra paleta de colores asegura una relación de contraste mínima de 4.5:1 para todo el texto y componentes críticos.

Lectores de pantalla

Etiquetas ARIA descriptivas y estructura semántica optimizada para JAWS, NVDA y VoiceOver.
